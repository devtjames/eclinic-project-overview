# eClinic – Online Medical Consultation System

eClinic is a web-based medical consultation platform that allows patients to book appointments, consult doctors, and receive prescriptions digitally.

This repository is a **case study and project overview**.  
The full source code is private.

---

## Project Overview

The goal of this project was to create a simple and reliable system for managing online medical consultations between doctors and patients.

The system supports three main user roles:

- Admin
- Doctor
- Patient

Patients can schedule appointments, doctors conduct consultations and issue prescriptions, and administrators manage the platform.

The focus of the project was clear workflows, reliable data handling, and a clean interface that works well on both desktop and mobile devices.

## System Roles

### Admin
- Manage doctors
- Manage patients
- View all appointments
- Monitor consultations

- View assigned appointments
- Conduct consultations
- Create prescriptions
- View consultation history

### Patient
- Register and login
- Book appointments
- View consultation history
- Print prescriptions
