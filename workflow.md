# System Workflow

This document describes how users interact with the eClinic system.

---

## Patient Registration

1. Patient opens the registration page
2. Patient enters personal information
3. Account is created
4. Patient can log in

---

## Appointment Booking

1. Patient logs into the system
2. Patient chooses a doctor
3. Patient selects date and time
4. Appointment request is created
5. Appointment status is saved as **pending**

---

## Appointment Management

Doctor or admin reviews the appointment.

Possible outcomes:

- Confirm appointment
- Reschedule appointment
- Cancel appointment

Once confirmed, the appointment appears in the doctor dashboard.

---

## Consultation

1. Doctor opens the scheduled appointment
2. Doctor starts consultation
3. Doctor records consultation notes
4. Consultation record is saved

---

## Prescription Creation

After the consultation:

1. Doctor writes a prescription
2. Medication and dosage are recorded
3. Prescription is stored in the database
4. Patient can access it from prescription history

---

## Prescription Viewing

Patient actions:

1. Login to dashboard
2. Open prescription history
3. View prescription details
4. Print prescription if needed

---

## Dashboard Overview

Each user sees different information.

### Admin Dashboard

- Total patients
- Total doctors
- Total appointments
- Recent activity
- Calendar overview

### Doctor Dashboard

- Upcoming appointments
- Patient consultations
- Prescription creation
- Calendar view

### Patient Dashboard

- Upcoming appointments
- Appointment booking
- Prescription history
- Calendar view
